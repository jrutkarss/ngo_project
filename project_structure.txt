# NGO Management System

## Project Structure

```
ngo_management_system/
│
├── docs/                  # Documentation folder
│   └── README.md          # Project documentation
│
├── src/                   # Source code folder
│   ├── main/              # Main application code
│   │   ├── controllers/    # Controllers for handling requests
│   │   ├── models/         # Database models
│   │   ├── routes/         # API routes
│   │   └── views/          # Views for the application
│   │
│   ├── config/           # Configuration files
│   │   └── config.js      # Application configuration
│   │
│   └── tests/            # Test files
│       ├── unit/           # Unit tests
│       └── integration/     # Integration tests
│
├── package.json           # Node.js package file
├── .gitignore              # Git ignore file
└── README.md              # Project overview
```

## Description
This project aims to provide a comprehensive system for managing NGO activities, including managing donations, volunteers, and events. The structure is designed to allow easy navigation and scalability as the application grows.
